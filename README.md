# ✈️ TripToDo

## Sobre o Projeto

O **TripToDo** é um sistema de planejamento de viagens desenvolvido para auxiliar os usuários na organização completa de suas viagens em um único ambiente.

A plataforma permite centralizar informações importantes, como destinos, roteiros, voos, hospedagens, transfers e restaurantes, evitando o uso de várias ferramentas diferentes, como blocos de notas, planilhas e aplicativos.

## Como Instalar

### 1. Clonar o repositório

```bash
git clone [https://github.com/seu-usuario/TripToDo.git](https://github.com/StefaniPm/TripToDo.git)
```

### 2. Entrar na pasta do projeto

```bash
cd TripToDo
```

### 3. Abrir o projeto na IDE

Recomenda-se utilizar:

- VS Code

### 4. Instalar as dependências

As dependências são gerenciadas pelo Maven:

```bash
mvn clean install
```

### 5. Executar a aplicação

```bash
mvn spring-boot:run
```

Ou execute a classe principal do projeto pela IDE.

## Funcionalidades

- Cadastro de usuários;
- Login de usuários;
- Criação de viagens;
- Edição e exclusão de viagens;
- Cadastro de destinos;
- Organização de roteiros por dia;
- Cadastro de atividades;
- Gerenciamento de voos;
- Gerenciamento de hospedagens;
- Gerenciamento de transfers;
- Gerenciamento de restaurantes;
- Visualização completa do planejamento da viagem.
